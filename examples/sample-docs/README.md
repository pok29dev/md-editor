# Sample Docs

โฟลเดอร์ตัวอย่างสำหรับทดสอบ MD Editor

## วิธีใช้

1. เปิดแอป → **Open Folder**
2. เลือกโฟลเดอร์ `examples/sample-docs`
3. คลิกไฟล์ใน sidebar เพื่อเปิด

## ไฟล์

| ไฟล์ | คำอธิบาย |
|------|----------|
| `README.md` | ไฟล์นี้ |
| `getting-started.md` | เอกสารเริ่มต้น |
| `guides/basics.md` | ตัวอย่าง nested folder |
